# 🚀 CareerPilot AI

CareerPilot AI is a student-focused career guidance platform designed to help users explore career opportunities, learning roadmaps, industry insights, and useful learning resources.

## 🌟 Features

### 🎯 Career Guidance

* Explore popular technology career paths.
* Learn required skills and salary ranges.
* Discover opportunities in different domains.

### 📚 Learning Roadmaps

* Step-by-step learning paths for:

  * Frontend Development
  * Data Analytics
  * Artificial Intelligence
  * Cybersecurity

### 💼 Industry Insights

* Understand current industry trends.
* Explore future career opportunities.
* Learn about in-demand technologies.

### 🚀 Projects Hub

* Project ideas for portfolio building.
* Beginner to advanced project suggestions.
* Helps students gain practical experience.

### 📖 Learning Resources

* Free Courses
* YouTube Channels
* Practice Websites
* Certification Resources

## 🛠️ Technologies Used
